# Zevi的小红书图文内容生成工具 - Demo生成提示词

## 项目概述
这是一个专为个人创作者设计的PC端小红书图文内容生成工具，集成了AI灵感推荐、内容创作、素材管理和个人中心功能。

## 完整Demo生成提示词

### 主应用架构提示词

```
创建一个基于React 18 + TypeScript + Vite的桌面端小红书内容创作工具，使用TailwindCSS进行样式设计。

项目结构：
- 使用React Router进行页面路由管理
- 使用Zustand进行状态管理
- 使用IndexedDB (通过Dexie.js)进行本地数据存储
- 使用TipTap实现富文本编辑器
- 集成OpenAI API进行AI内容生成

主要页面：
1. 首页导航 - 包含四个主要功能入口的卡片式布局
2. 找灵感页面 - 关键词输入和AI灵感推荐展示
3. 来创作页面 - 小红书风格的内容编辑器
4. 看素材页面 - 素材库管理和预览
5. 我自己页面 - 个人信息和创作统计

设计要求：
- 小红书品牌色彩：主色调#FF2442，辅助色为白色和浅灰色
- 圆角卡片设计，阴影效果，现代简洁风格
- 响应式布局，适配PC端使用
- 平滑的页面过渡动画和交互反馈
```

## 开发建议

### 开发顺序建议
1. 先搭建基础项目结构和路由
2. 实现UI组件库和基础样式
3. 开发数据管理层（IndexedDB + Store）
4. 按页面顺序实现功能：首页 → 找灵感 → 来创作 → 看素材 → 我自己
5. 集成AI功能和优化用户体验
6. 添加性能优化和错误处理

### 技术选型说明
- **React + TypeScript**：提供类型安全和现代开发体验
- **Vite**：快速的构建工具和开发服务器
- **TailwindCSS**：高效的样式开发
- **Zustand**：轻量级状态管理
- **TipTap**：功能强大的富文本编辑器
- **Dexie.js**：简化的IndexedDB操作

### 扩展性考虑
- 模块化架构便于功能扩展
- 插件化设计支持自定义功能
- 主题系统支持个性化定制
- API抽象层便于切换AI服务提供商
